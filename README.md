# 🐾 PetCareStay

## Sistem Informasi Penitipan Hewan Peliharaan Berbasis Web

PetCareStay adalah aplikasi berbasis web yang dibuat menggunakan **Python Flask** sebagai proyek UAS mata kuliah Pengantar Pemrograman. Aplikasi ini digunakan untuk membantu pengelolaan data penitipan hewan peliharaan, mulai dari data pemilik, data hewan, data booking, hingga data kontak.

---

## 📌 Fitur

- Login Admin
- Dashboard Admin
- CRUD Data Pemilik
- CRUD Data Hewan
- CRUD Data Booking
- CRUD Data Kontak
- Dashboard Statistik
- Database SQLite

---

## 🛠️ Teknologi yang Digunakan

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- SQLite
- HTML
- CSS
- Bootstrap Icons

## 💾 Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/USERNAME/PetCareStay.git
```

### 2. Masuk ke Folder Project

```bash
cd PetCareStay
```

### 3. Aktifkan Virtual Environment

Windows

```bash
venv\Scripts\activate
```

### 4. Install Library

```bash
pip install -r requirements.txt
```

### 5. Jalankan Program

```bash
python app.py
```

Buka browser:

```
http://127.0.0.1:5000
```
